# Prerequisites

Before you start, make sure you have the following:

## Required

- **A free OpenWeatherMap account** — [Sign up here](https://home.openweathermap.org/users/sign_up)
- **An API key** — Generated after sign-up (takes up to 2 hours to activate)
- **A terminal or command-line tool** — To run `curl` commands or scripts
- **Node.js 18+** or **Python 3.8+** — For the tutorial code samples

## Recommended

- **A code editor** — VS Code, Sublime Text, or any editor you prefer
- **Postman** (optional) — For testing API requests interactively
- **Basic knowledge of REST APIs** — Understanding of HTTP methods, status
  codes, and JSON

## Check your setup

Verify Node.js is installed:

```bash
node --version
# Expected output: v18.x.x or higher
```

Verify Python is installed:

```bash
python3 --version
# Expected output: Python 3.8.x or higher
```

If either command fails, install the missing runtime before proceeding.
